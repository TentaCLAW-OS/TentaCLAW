A node is any physical or virtual machine running the TentaCLAW agent. A single server with 8 GPUs counts as one node, not eight. This keeps pricing predictable regardless of GPU density. Ephemeral cloud-burst nodes (RunPod, Lambda) are billed pro-rata by the hour when active.
