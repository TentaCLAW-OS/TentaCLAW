How It Works

Three steps to your own AI cloud.

No Kubernetes. No Docker compose files. No YAML hell. Just hardware and an octopus.

1

Boot It

Flash USB or run the one-liner. TentaCLAW auto-detects your GPUs, CPU capabilities, and available VRAM.

curl -fsSL tentaclaw.io/install | bash
2

Nodes Find Each Other

mDNS auto-discovery on LAN. New machines join the cluster in seconds. No IPs. No config. No ports.

tentaclaw: 3 nodes discovered
3

Deploy and Route

Deploy models, CLAWtopus routes everything. VRAM-aware scheduling. Smart load balancing. It just works.

clawtopus deploy llama3.1:70b
