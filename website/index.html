Features

Everything you need.
Nothing you don't.

From single GPU to multi-node cluster. TentaCLAW handles discovery, scheduling, routing, and monitoring so you can focus on building.

Zero Config

Plug in a GPU. TentaCLAW detects it, configures it, and starts serving inference. No YAML. No manifests. Just GPUs.

Multi-Node Clusters

Distribute models across any number of machines. Automatic load balancing, failover, and tensor parallelism built in.

Live Dashboard

Proxmox-style command center. Every GPU, every model, every request visible in real-time with sparklines and status pills.

AI Chat Built-in

Chat with any model in your cluster directly from the dashboard. OpenAI-compatible API included for drop-in integration.

Smart Scheduling

Automatic model placement, VRAM optimization, and cold-start minimization. The right model on the right GPU, always.

Open Source

MIT licensed. No vendor lock-in. No telemetry. Your GPUs, your data, your rules. Fork it, extend it, own it.
