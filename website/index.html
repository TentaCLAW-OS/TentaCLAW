What TentaCLAW does

Zero Config

Plug in a machine, boot it up. TentaCLAW detects GPUs, finds the gateway, and starts serving inference. No config files, no ports, no IPs.

Smart Load Balancing

Requests route to the best node automatically. VRAM-aware, circuit breaker, auto-retry on failure. Your cluster acts as one.

Self-Healing

5-level watchdog: restart service, kill process, reset GPU, reboot. If something breaks at 3am, TentaCLAW fixes it.

OpenAI Compatible

Drop-in replacement for OpenAI API. Function calling, JSON mode, streaming, embeddings. Point any client at your cluster.

Model Management

Search models, check VRAM fit, deploy to best node. Auto mode lets the system decide everything for you.

Real-Time Dashboard

GPU temps, VRAM bars, inference flow, node health. Big numbers, live data. See everything at a glance.
