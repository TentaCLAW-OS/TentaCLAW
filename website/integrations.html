OpenAI API Full Compat
Complete compatibility with the OpenAI SDK. Streaming, function calling, JSON mode, vision, embeddings -- everything works out of the box.

python
# Drop-in replacement -- just change the base URL
from openai import OpenAI

client = OpenAI(
    base_url="http://your-cluster:8080/v1",
    api_key="your-key"
)

response = client.chat.completions.create(
    model="llama3.1:70b",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)

Built-In Coding Agent

Your Cluster Has a Brain.

A full AI coding agent built into the CLI. No dependencies. Persistent memory. Works everywhere your cluster runs.

  • Zero dependencies -- pure Node.js, no Python, no pip, no conda
  • 12 built-in tools -- read_file, write_file, list_dir, run_shell, search_files, edit_file, and more
  • Persistent sessions -- JSONL event logs, resumable via --resume <id>
  • Workspace memory -- SOUL.md, USER.md, MEMORY.md, IDENTITY.md for persistent context
  • 20+ slash commands -- /quit /new /status /sessions /save /export /clear /compact /model /think /workspace
  • Cross-platform -- Linux, macOS, Windows, Proxmox. Shell shortcut: ! git status
