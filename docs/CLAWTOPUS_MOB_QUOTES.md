## Usage Notes

- **Rotation**: Never show the same line twice in a row. Implement a rotation pool per category.
- **Tone matching**: Match severity to tone. Swagger lines for healthy states. Threat lines for warnings. Wise guy lines for general CLI output.
- **Context awareness**: Error lines should still communicate the actual error first (per WAITING_STATE_COPY.md guidance: status first, joke second).
- **Voice consistency**: CLAWtopus is always in control. Even when reporting errors, she's not panicking -- she's identifying who messed up.
- **Personalization**: Replace "Node-07" with actual node names for maximum impact.
