## Developer Secrets

### Kernel Params
- `tentaclaw.meme=on` - Enable meme mode
- `tentaclaw.verbose=MAX` - Maximum CLAWtopus quotes
- `tentaclaw.colors=RAINBOW` - Extra colorful output

### Debug Mode
```bash
# Enable debug logging
export TENTACLAW_DEBUG=1

# Show hidden network nodes
export TENTACLAW_SHOW_ALL=1

# Force CLAWtopus ASCII mode
export TENTACLAW_ASCII_MODE=1
```
